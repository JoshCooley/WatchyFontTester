# WatchyFontTester
Watchy Watchface for font testing

## Requirements
* Pre-configured Arduino IDE
* macOS (assumed in make.sh)

## To build:
* Run ./make.sh. This will:
  - Patch the font in TesterFont.fontsrc/TesterFont.cpp for use with Watchy
  - Build and upload the watchface to your Watchy.

*Test font built with adafonteditor: <https://github.com/w3woody/adafonteditor>*
